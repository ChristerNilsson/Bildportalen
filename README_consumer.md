# Bildportalen: användare

Bildportalen visar schackbilder från anslutna fotografer.

## Söka

Skriv i sökrutan högst upp. Sökningen matchar både kataloger och filnamn.

Du kan söka på flera ord. Om du söker på två termer visas träffar som matchar båda före träffar som bara matchar den ena.

Exempel:

```text
Tjejträffen prisutdelning
Bo_Ek
Wasa 2026
```

## Kataloger

Katalogknapparna visar hur många bilder som finns i varje katalog. Klicka på en katalog för att gå in i den. Klicka på `Upp` för att gå tillbaka till föräldrakatalogen.

Länkar till Medlemssystemet, Chess-Results, Vimeo, pdf, txt och url-filer visas ovanför katalogknapparna när de hör till aktuell katalog.

## URL

URL:en uppdateras när du söker eller byter katalog. Du kan kopiera URL:en för att dela samma vy med någon annan.

## Visa och hämta bild

Klicka på en bild för att öppna den i Googles visningsprogram. Där kan du zooma, panorera och hämta bilden om du har behörighet.
